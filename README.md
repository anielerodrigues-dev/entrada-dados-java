# 📋 Entrada de Dados — Java

Programa simples em Java que coleta o nome e a idade do usuário via terminal e exibe uma mensagem personalizada de boas-vindas.

## 📖 Sobre o projeto

Este projeto foi desenvolvido como parte dos meus estudos em Java.

O objetivo foi praticar:

- Entrada de dados via `Scanner`
- Tipos primitivos (`int`) e `String`
- Concatenação de strings
- Estrutura básica de um programa Java (`main`)

## 🛠 Tecnologias

- Java

## 🎯 Objetivo

Praticar os fundamentos da linguagem Java e construir uma base sólida para projetos maiores.

## 🚀 Como executar o projeto

Você vai precisar ter o **JDK** instalado (Java 8 ou superior).

```bash
# Clone este repositório
git clone https://github.com/seu-usuario/entrada-dados-java.git

# Acesse a pasta do projeto
cd entrada-dados-java/src

# Compile o programa
javac EntradaDados.java

# Execute o programa
java EntradaDados
```
